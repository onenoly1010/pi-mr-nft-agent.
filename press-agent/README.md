# OINIO Automated Press Agent

Generates announcements for OINIO milestones. 

## Usage

python press-agent/press_release_generator.py

## Milestones

1. contracts_ready - Smart contracts finalized
2. deployment_complete - Deployed to Pi Mainnet
3. first_mint - 6 seed models minted

## Outputs

Saved to press-agent/outputs/TIMESTAMP/
